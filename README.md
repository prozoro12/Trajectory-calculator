# Trajectory Calculator

A simple Python program to calculate and visualize the trajectory of an object launched at a specified initial velocity and angle.

This project computes the motion of a projectile under gravity using basic physics formulas, and can output key metrics such as flight time, range, and maximum height.

## Features

- Calculates the **trajectory** of a projectile given:
  - Initial velocity
  - Launch angle
  - Optional initial height
- Computes:
  - Total time of flight
  - Maximum height
  - Horizontal range
- (Optional) Plots the trajectory using `matplotlib`
